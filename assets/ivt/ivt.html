INTRODUCTION

Linux kernel, the most successful open-source product, is made virtualization capable with a simple kernel module called Kernel-based Virtual Machine (KVM). KVM is perceived as a very promising hypervisor by users and vendors and hence going to be a very prominent candidate for virtualization and cloud in the near future. This contest is conducted by IBM to help participants improve their Linux and KVM skills.

TUTORIALS

Participants are directed to get Fedora 17 to a VT-enabled x86 system and try out KVM. You can get the iso images for that from the following links:

1.http://download.fedoraproject.org/pub/fedora/linux/releases/17/Live/i686/Fedora-17-i686-Live-Desktop.iso

2.http://download.fedoraproject.org/pub/fedora/linux/releases/17/Fedora/i386/iso/Fedora-17-i386-DVD.iso

Participants can try out following scenarios with Linux and KVM:
  • KVM setup
  • Create Virtual Machines
  • Start, shutdown, reboot VM's
  • Configure networking between host and guest.
  • Configure public bridge network
