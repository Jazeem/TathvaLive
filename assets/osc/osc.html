Problem statement

The objective is to design and implement an analog/digital circuit to play the specified game on the display of the Cathode Ray Oscilloscope by controlling the input signals to the CRO.
Schematic View of Game on CRO display :

Specifications

  1. Basic Game Description:
    • The game should contain THREE parallel paths.
    • On each of the path, randomly moving obstacles are to be generated.
    • The player's initial position should be below the THREE paths and final position above all the paths.
    • It should be manually controlled in such a way that the character should cross all the roads without hitting any obstacles.
  2. Optional Features:
    • On hitting the obstacle, the game should be reset.
    • Additional levels, with increased obstacle speed or increased parallel paths.
    • Display of time elapsed

Event Rules

  • On hitting any of the obstacles, an indication should be given(like LED on or a BEEP sound).
  • The character should move only in the vertical direction.
  • The character should not rest between the parallel paths.
  • The input control should be given through push buttons.
  • A function generator, a variable DC source (0-15 V) and an oscilloscope will be provided.


*Note that, though the extra features contain a significant portion of allocated marks, they would be taken into consideration only when the participant has fully implemented the basic specifications.

Judging Criteria

  • Simplicity of the circuit will be taken under consideration.
  • Complexity in the game will be considered.
  • Circuit without micro-controllers would be preferred.
  • Judges decision is final.
